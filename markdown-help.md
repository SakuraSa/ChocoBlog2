##ChocoBlog是什么?
一个Markdown内容发布系统

##如何使用Markdown语法？

* 标题样式

    ```
    # 标题
    ## 标题
    ### 标题
    ```
    
* 列表样式

    ```
    * 父列表1
        * 子列表1
        * 子列表2
        * 子列表3
    * 父列表2
    ```

* 链接样式

    ```
    [链接显示的文字](http://www.google.com)
    ```
    
* 图片样式

    ```
    ![图片名称](/static/image/avatar.jpg)
    ```

* 更多样式

    更详细的Markdown语法请参见[Markdown: Syntax](http://markdown.tw/)
