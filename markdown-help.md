##ChocoBlog
一个Markdown内容发布系统


##Markdown功能

* 链接样式

    [Google](https://www.google.com)
    
* 图片样式

    ![My avatar](/static/image/avatar.jpg)

* 代码高亮
    
    ```python
    
    import this
    
    if __name__ == '__main__':
        print('This is Python.')
    
    ```

* 表格
    
    | Header 1 | *Header* 2 |
    | -------- | -------- |
    | `Cell 1 | [Cell 2](http://example.com) link |
    | Cell 3 | **Cell 4** |
    
* 公式

    当 \\(a \ne 0\\), 方程 \\(ax^2 + bx + c = 0\\) 存在两个解：
    $$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
