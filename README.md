#ChocoBlog2

Flask based blog for myself.

##Setup

1. run ```python ChocoBlog.py -s -c``` to init database
2. run ```python ChocoBlog.py --host=localhost --port=80``` to start server

##Requirements


1. Flask=0.10.1
2. SQLAlchemy==1.0.10
